# Gateway endpoints

The gateway's endpoint detail is auto-generated at every docs build.

- **[Master API catalogue](/reference/api-catalogue)** — every endpoint
  across every service in one searchable table (gateway + 4 parsers).
- **[Gateway OpenAPI page](/reference/openapi/gateway)** — gateway-only
  endpoint list with request/response schemas.

The OpenAPI source itself lives at
[`http://localhost:8000/openapi.json`](http://localhost:8000/openapi.json)
when the gateway is running, with an interactive Swagger UI at
[`http://localhost:8000/docs`](http://localhost:8000/docs).
