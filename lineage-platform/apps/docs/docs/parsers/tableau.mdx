## Pipeline

1. **Container** — `.twbx` is a ZIP holding a `.twb` and any extracts.
   `parser/workbook.py` unzips into a temp dir.
2. **DOM** — `.twb` is XML; lxml parses it into a DOM the `extractor/`
   package walks.
3. **Extract** — `<datasource>` blocks yield connections, tables, fields,
   joins, calculated-field expressions. `<worksheet>` and `<dashboard>`
   bind data fields to charts via field-usage records.
4. **Custom-SQL** — calculated-field expressions and embedded `<query>`
   blocks route through sqlglot to produce `CTEColumnIR` rows.
5. **IR** — `models/domain.py` typed dataclasses: `WorkbookIR`,
   `DatasourceIR`, `WorksheetIR`, `DashboardIR`, `ParameterIR`,
   `CTEColumnIR`.
6. **Write** — `graph/writer.py` emits `:Workbook`, `:Datasource`,
   `:Worksheet`, `:Dashboard`, `:Attribute`, `:Table`, `:Connection`,
   `:Parameter` plus the `USES_DATASOURCE`, `HAS_ATTRIBUTE`, `READS_TABLE`,
   `DERIVES_FROM` edges.
