## What each parser tests

- **Unit** — pure-IR tests of the visitor + writer logic; no Neo4j.
- **Integration** — round-trip via testcontainers-Neo4j: parse →
  MERGE → re-query → assert.
- **Idempotency** — parse twice, assert node-set is byte-identical.
- **Real-world fixtures** — production-shaped inputs (large composer
  dumps, real `.twb` exports).

Per-parser detail:

- [Tableau](/tests/tableau)
- [TWS](/tests/tws)
- [QlikView](/tests/qlikview) (includes pass / xfail breakdown from `TEST_REPORT.md`)
- [Spark](/tests/spark) (includes the v0.2 feature matrix from `FEATURES.md`)

## How to run

```bash
# Single parser
cd tableau-parser && .venv/bin/pytest

# Inside docker (no local venv needed)
docker exec lineage-tableau-parser pytest tests/unit
```
